Detailed Description

Greedy graph-coloring register allocator.

Phase 1 — Simplification: Repeatedly find a node with degree < N, remove it from the active graph and push it onto a stack. If no such node exists, pick one to spill (lowest degree heuristic), mark it as COLOR_SPILLED, and remove it. Repeat until the graph is empty.

Phase 2 — Coloring: Pop each node from the stack, reactivate it, and assign the lowest available color not used by any active neighbour.

Note
Colors are 0-indexed (r0, r1, ..., r(N-1)). Spilled webs are marked with COLOR_SPILLED (-2) → output as "M".
