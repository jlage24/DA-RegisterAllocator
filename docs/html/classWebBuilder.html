Detailed Description

Builds Webs from a list of LiveRanges.

A Web is the union of one or more live ranges of the same variable that must be merged according to the following rules:

  1. Two ranges of the same variable share at least one program line.
  2. One range ends at line X with marker '-' and another begins at line X with marker '+' (same variable — e.g. i = i + 1).

The merging is iterative: after each fusion, the loop restarts because a new merged group may now share lines with another group.

Note
Time Complexity: O(R^3 * L) worst case, where R = number of live ranges and L = average lines per range. Acceptable for typical compiler inputs where R and L are small.
