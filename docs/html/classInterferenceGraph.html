Detailed Description

Interference Graph for register allocation.

Extends Graph<int> where each node's key is a webId. Edges are undirected (added in both directions). Provides graph-coloring and degree utilities needed by the allocation algorithms.

Note
The base Graph<int> stores directed edges. We simulate undirected behaviour by always adding edges in both directions via addUndirectedEdge().
