Detailed Description

Register allocator with intentional pre-spilling (T2.2).

Before running the greedy coloring, intentionally spills the K webs with the most program lines covered. This reduces the interference graph and increases the chance that the remaining webs fit in N registers.

Spill heuristic: largest number of lines covered (most live = most likely to interfere with others, so most beneficial to remove).

Note
After pre-spilling, runs the same greedy coloring as basic (T2.1).
